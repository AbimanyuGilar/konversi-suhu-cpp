2. Jika input besar suhu dari pengguna tidak valid, maka sistem akan kembali ke langkah input besar suhu
3. Jika input besar suhu valid, pengguna lanjut memilih metode konversi
4. Jika input metode konversi dari pengguna tidak valid, maka sistem akan kembali ke langkah input metode konversi
5. Jika input metode knversi valid, sistem mengkonversi suhu
6. Sistem menampilkan hasil konversi
7. Selesai
